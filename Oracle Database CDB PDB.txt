프로파일 생성과 적용 (SQL)
Connect as ADMIN: Connect to your PDB as a user with CREATE PROFILE and ALTER PROFILE privileges (e.g., ADMIN).
Create the Profile: Define the limits using CREATE PROFILE.
sql
CREATE PROFILE app_user_profile LIMIT
  SESSIONS_PER_USER 3
  CPU_PER_SESSION UNLIMITED
  IDLE_TIME 60
  PASSWORD_LIFE_TIME 90
  PASSWORD_LOCK_TIME 5
  PASSWORD_REUSE_MAX 10;
This creates a profile named app_user_profile with specific resource and password constraints.
Assign the Profile to a New User: Create the user and apply the profile.
sql
CREATE USER new_user IDENTIFIED BY "Password123#"
  PROFILE app_user_profile;
Grant Session Privileges: Allow the new user to connect.
sql
GRANT CREATE SESSION TO new_user;
