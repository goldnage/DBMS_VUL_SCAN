<<MySQL root password reset in Ubuntu-24.04>>

1. MySQL 서비스 중단
   $ sudo systemctl stop mysql
   
2. 안전 모드로 MySql 시작

   만약 없다면 필요한 실행 디렉토리 생성
   
   $ sudo mkdir -p /var/run/mysqld
   $ sudo chown mysql:mysql /var/run/mysqld
   
   백그라운드로 서버를 시작
   
   $ sudo msyqld_safe --skip-grant-tables --skip-networking &
   
3. 루트로 로그인 후 패스워드 재설정

   $ mysql -u root
   
   grant tables 리로드
   SQL> flush privileges;
   
   루트 패스워드 갱신
   SQL> alter user 'root'@'localhost' identified with caching_sha2_password by 'your_new_password';

4. MySQL 재시작 

   안전 모드 프로세스 중지
   $ sudo pkill mysqld
   
   정상 서비스 시작
   $ sudo systemctl start mysql
   
5. 검증

   $ mysql -u root -p
